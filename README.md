# kv2 = koa 2.x + vue 2.x server side rendering

> A Vue.js project with vue 2.0, vue-router and vuex starter kit for server side rendering.

## Build Setup

Node.js 4.x+

``` bash
npm install
npm run build
npm start
```

## Development Setup

```bash
npm install
npm run dev
```
